# Loto by SdS v4.1.9

Cette version simplifie l'affichage des lots sur l'écran public.

- Carton plein : `1er LOT : ...`, `2e LOT : ...`, `3e LOT : ...`.
- Jeu à la ligne : ajout de `1 LIGNE`, `2 LIGNES`, `CARTON PLEIN` sur les lignes correspondantes.
- Le libellé reste blanc et le nom du lot est affiché en orange.
