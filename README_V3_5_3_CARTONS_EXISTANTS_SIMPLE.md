# V3.5.3 - Cartons existants simplifie

Dans Administration > Cartons :

1. Section Creation de cartons conservee.
2. Section Ajouter un carton existant simplifiee :
   - Identifiant du carton uniquement.
   - Saisie des 15 numeros.
   - Bouton Controler.
   - Affichage du carton reconstruit en 3 x 9 avec des 0 pour les cases vides.
   - Bouton Enregistrer apres controle.

Le numero interne, la serie et le statut sont geres automatiquement.
Apres enregistrement, les champs sont vides pour saisir le carton suivant.
