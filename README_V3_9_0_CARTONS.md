# Loto by SdS v3.9.0

## Nouveautés

- La création est exprimée en nombre de pages.
- 4 cartons individuels A6 sur A4, sans traits de découpe.
- Planche de 4 sur A4 avec un QR code commun.
- Planche de 6 + 2 cartons individuels sur A3 : un QR pour les 6 cartons et un QR par carton individuel.
- Planche de 8 sur A3 avec un QR code commun.
- Le scanner Vente vend ou retourne tous les cartons d’une planche en un seul scan.
- Réimpression d’un carton ou d’une plage sans modifier les données ni l’état de vente.
- Archivage et restauration d’un carton ou d’une plage. Les numéros ne sont jamais réutilisés.

## Mise en production

Aucune nouvelle table Supabase n’est nécessaire par rapport à la v3.8.1. Remplacer les fichiers du dépôt GitHub par ceux de cette archive.
