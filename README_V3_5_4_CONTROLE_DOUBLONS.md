# V3.5.4 - Contrôle cartons existants

Dans Administration > Cartons :

1. Saisir l’identifiant imprimé du carton.
2. Saisir les 15 numéros.
3. Cliquer sur **Contrôler**.
4. Si la saisie est cohérente, l’aperçu 3 x 9 est affiché avec les 0 dans les cases vides.
5. Cliquer sur **Enregistrer**.

À l’enregistrement, l’application vérifie :
- identifiant déjà utilisé ;
- doublon des 15 numéros déjà présents en base ;
- validité de la grille.
