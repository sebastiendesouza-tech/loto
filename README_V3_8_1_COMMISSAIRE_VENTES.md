# Loto by SdS v3.8.1 — Commissaire et suivi des ventes

La page Commissaire indique désormais clairement le mode du loto actif :

- **Suivi des ventes activé** : le gain est contrôlé et le carton doit avoir été vendu pour ce loto.
- **Suivi des ventes désactivé** : seul le gain est contrôlé. Cette configuration convient aux cartons existants sans QR code, dont le numéro peut être saisi manuellement.

Le scanner QR reste disponible pour les cartons qui en possèdent un.
