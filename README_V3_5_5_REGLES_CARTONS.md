# V3.5.5 - Règles cartons corrigées

Correction de la gestion des cartons dans Administration > Cartons.

Règles appliquées :

- 15 numéros exactement.
- 5 numéros par ligne.
- Numéros entre 1 et 90.
- Aucun doublon dans le carton.
- Chaque ligne doit être croissante de gauche à droite.
- Chaque numéro est placé automatiquement dans sa colonne de dizaine.
- Une colonne peut être vide.
- Une colonne peut contenir 1, 2 ou 3 numéros.
- Aucun contrôle de croissance verticale dans les colonnes.

Le générateur de cartons a été corrigé avec les mêmes règles : il ne force plus au moins un numéro par colonne.
