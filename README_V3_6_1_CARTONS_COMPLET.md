# V3.6.1 - Cartons complet

Onglet Administration > Cartons organisé en trois sections :

1. Création de cartons
2. Ajout d’un carton existant
3. Ajout d’une planche de 6 cartons existants

La saisie unitaire a été remise entre la création et la saisie de planche.

Règles conservées :
- 15 numéros par carton ;
- 5 numéros par ligne ;
- lignes strictement croissantes ;
- colonnes selon les dizaines ;
- colonnes pouvant être vides ;
- pas de croissance verticale imposée ;
- contrôle des 90 numéros uniques sur une planche de 6.
