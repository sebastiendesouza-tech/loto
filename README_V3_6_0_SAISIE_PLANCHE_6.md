# V3.6.0 - Saisie d'une planche de 6 cartons

Dans Administration > Cartons :

- ajout d'une section pour saisir une planche de 6 cartons ;
- un identifiant par carton ;
- une ligne de saisie par carton avec les 15 numéros ;
- contrôle individuel de chaque carton ;
- contrôle global : les 90 numéros doivent être présents une seule fois sur la planche ;
- aperçu des 6 cartons en 3 x 9, affichés 2 par ligne ;
- enregistrement des 6 cartons en une seule opération ;
- vidage automatique après enregistrement.
