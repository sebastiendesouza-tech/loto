# V3.5.2 - Onglet Cartons simplifié

Dans Administration > Cartons, il reste uniquement :

1. Créer des cartons
2. Ajouter un carton existant

La section Ajouter un carton existant demande :
- l'identifiant imprimé du carton ;
- les 15 numéros saisis rapidement ;
- la série et le statut.

La grille Supabase est reconstruite en 3 x 9 avec des 0 dans les cases vides, afin que l'affichage corresponde au vrai carton papier.

Le scan reste réservé au commissaire.
